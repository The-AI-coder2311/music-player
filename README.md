# music-player
# Music Player Pro (Tkinter + VLC)

A desktop music player built with Python, featuring:
- Playlists system
- Queue system
- Drag & drop audio import
- Search bar
- Right-click song actions
- Playback progress slider (seek)
- Volume control
- VLC-based audio engine

---

## Features

- Create / delete playlists
- Add / remove songs
- Drag and drop audio files
- Search songs instantly
- Right-click menu:
  - Play
  - Add to queue
  - Remove song
- Queue system
- Progress bar (seek slider)
- Volume slider
